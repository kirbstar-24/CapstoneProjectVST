This is a VST3 FX plugin developed with JUCE.

To build and use in Windows:
1. Download Projucer
2. Open the .jucer file with Projucer and open in Visual Studios
3. Build the solution 
4. Navigate to the builds file to the VST file
5. Builds -> VisualStudio -> x64 -> Debug -> VST3 -> NewProject.vst3 -> x86_64-win -> NewProject.vst3
6. Copy the .vst3 file to your VST folder for your DAW (Defualt is located in Program Files -> Common Files -> VST3
7. Open and enjoy in your favorite DAW!

*Note this project uses JUCE an open source library: https://github.com/juce-framework/JUCE
*This project is likely to be expanded upon in the future
